# Sistema de Gerenciamento de Tarefas Ágil

## Descrição

Este projeto foi desenvolvido para a disciplina de Engenharia de Software.

O objetivo é criar um sistema de gerenciamento de tarefas utilizando metodologias ágeis, GitHub Projects, GitHub Actions e testes automatizados.

## Tecnologias

- Python
- GitHub
- GitHub Actions
- Pytest
