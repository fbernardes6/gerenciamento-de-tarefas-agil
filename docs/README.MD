# Sistema de Gerenciamento de Tarefas Ágil

## Descrição
Este projeto foi desenvolvido em Python para gerenciar tarefas de forma simples utilizando o terminal.

## Funcionalidades

- Adicionar tarefa
- Listar tarefas
- Excluir tarefa
- Encerrar o sistema

## Tecnologias utilizadas

- Python 3
- Visual Studio Code
- Git
- GitHub

## Como executar

1. Abra o projeto no Visual Studio Code.
2. Abra o terminal.
3. Execute:


python src/main.py
